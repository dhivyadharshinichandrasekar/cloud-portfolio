// Jenkinsfile - SmartBuy CI/CD Pipeline
pipeline {
    agent any
    
    environment {
        AWS_REGION = 'ap-south-1'
        ECR_REPO = 'smartbuy-app'
        K8S_NAMESPACE = 'production'
    }
    
    stages {
        stage('Checkout') {
            steps {
                git branch: 'main', 
                    url: '[github.com](https://github.com/dhivyadharshinichandrasekar/smartbuy.git)'
            }
        }
        
        stage('SonarQube Analysis') {
            steps {
                withSonarQubeEnv('sonarqube') {
                    sh 'sonar-scanner'
                }
            }
        }
        
        stage('Docker Build & Push') {
            steps {
                sh 'docker build -t $ECR_REPO:$BUILD_NUMBER .'
                sh 'aws ecr get-login-password | docker login --username AWS --password-stdin $ECR_URL'
                sh 'docker push $ECR_URL/$ECR_REPO:$BUILD_NUMBER'
            }
        }
        
        stage('Deploy to Kubernetes') {
            steps {
                sh 'kubectl apply -f k8s/ -n $K8S_NAMESPACE'
                sh 'kubectl set image deployment/smartbuy smartbuy=$ECR_URL/$ECR_REPO:$BUILD_NUMBER -n $K8S_NAMESPACE'
            }
        }
    }
}

< Key Features />

Microservices Architecture

Decomposed monolithic e-commerce into 8 microservices: User Service, Product Catalog, Cart, Order, Payment, Inventory, Notification, and Search services.

Auto-Scaling

Implemented Horizontal Pod Autoscaler (HPA) with custom metrics, scaling based on CPU utilization and request rate for handling traffic spikes.

Blue-Green Deployment

Zero-downtime deployments using blue-green strategy with automatic rollback capabilities on health check failures.

Security Scanning

Integrated Trivy for container vulnerability scanning and SonarQube for static code analysis in the CI/CD pipeline.

Observability Stack

Full observability with Prometheus for metrics, Grafana dashboards, and ELK stack for centralized logging.

Infrastructure as Code

Complete AWS infrastructure provisioned using Terraform modules including VPC, EKS cluster, RDS, and ElastiCache.
